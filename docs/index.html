Inclusion matters!



In the digital era, everyone deserves equal access to the digital resources.

Our extensions generates text description for all the images you visit, allowing blind people to understand the images.

About

Increases Inclusivity

Our team prepared chrome extension that enables blind people to get more out of their website experience.

Fixing Bad Practises

We make sure to include the alt text in places creators don't do.

Deep Learning

We used deep learning model for generating the captions for those images to enrich user website browsing experience.


How to install

Checkout our GitHub repo: github.com/Ache17/oxford_hack

Download this .zip fie and extract it.

Go to chrome://extensions and check Developer Mode

Click Load Unpacked and select the unzipped folder

Enjoy the functionality!
